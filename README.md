**Neon Pulse Survivor** is a fast-paced, browser-based survival shooter set in a vibrant neon cyber arena. Players face endless waves of geometric enemies, testing their reflexes, strategy, and survival instincts. Built entirely with **HTML, CSS, and JavaScript**, this game features smooth controls, dynamic visuals, and an engaging gameplay loop.

## Features

* **Dynamic Waves:** Progressive enemy waves with increasing difficulty.
* **Varied Enemies:** Triangles (seeker) and hexagons (heavy) with unique behaviors.
* **Intuitive Controls:** Mouse/Touch movement or WASD keyboard support.
* **Auto Shooting:** Player fires automatically at the nearest enemy.
* **Health & Score HUD:** Track player health, wave number, and score in real-time.
* **Particle Effects:** Explosions and neon trails for a visually striking experience.
* **Responsive Design:** Works seamlessly across desktops and mobile devices.

## How to Play

* **Move:** Mouse / Drag / WASD
* **Shoot:** Automatic
* **Goal:** Survive as many waves as possible and achieve a high score!

## Tech Stack

* **HTML5 Canvas** for rendering
* **Vanilla JavaScript** for game logic
* **CSS3** for neon-themed UI and effects

## Getting Started

Simply clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/yourusername/neon-pulse-survivor.git
cd neon-pulse-survivor
open index.html
```
